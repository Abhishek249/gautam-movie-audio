# 🎬 गौतम — मुक्ति का मार्ग
## Antgaddashang Sutra Animated Movie — Audio Generator

**Live App:** https://abhishek249.github.io/gautam-movie-audio

Generate all narration and dialogue audio for the animated movie based on the first chapter (Varga 1, Adhyayan 1) of the Antgaddashang Sutra — the 8th Anga of the Jain Agamas.

### Features
- 🎙️ 22 voice segments across 13 scenes
- 4 distinct voices: Narrator (onyx), Gautam (ash), Dharini (nova), Sudharma (sage)
- Powered by OpenAI `gpt-4o-mini-tts` with custom Hindi voice directions
- One-click generate all · Play preview · Download MP3

### Usage
1. Get an OpenAI API key from https://platform.openai.com
2. Enter it in the app
3. Click "⚡ सभी Generate करें"
4. Download MP3s → attach to Canva slides → export MP4

### Tech
Vite + React · OpenAI TTS API · Deployed via GitHub Pages
