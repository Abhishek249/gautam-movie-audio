# 🎬 गौतम — मुक्ति का मार्ग
## Antgaddashang Sutra Animated Movie — Audio Generator

**Live App:** https://abhishek249.github.io/gautam-movie-audio

Generate all narration and dialogue audio for the animated movie based on  
Varga 1, Adhyayan 1 of the Antgaddashang Sutra (8th Anga of Jain Agamas).

### Features
- 🎙️ 22 voice segments across 13 scenes
- 4 voices: Narrator (onyx), Gautam (ash), Dharini (nova), Sudharma (sage)
- Powered by OpenAI gpt-4o-mini-tts with Hindi voice directions
- Generate all · Play preview · Download MP3

### Usage
1. Get OpenAI API key from https://platform.openai.com
2. Enter in the app → click Generate → download MP3s
3. Attach to Canva slides → export MP4 🎬
